# some-scripts

Scripts utilitarios para baixar e inventariar samples do pacote Vengeance.

## Arquivos

- `download-vengeance-samples.ps1`: versao original em PowerShell.
- `download-vengeance-samples.sh`: versao em shell com inventario, listagem de faltantes e confirmacao antes do download.

## Uso rapido

```bash
./download-vengeance-samples.sh --inventory-only
./download-vengeance-samples.sh
./download-vengeance-samples.sh --yes
```
